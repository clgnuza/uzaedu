Yerel -> canli saglikli akis
=====================

AYRIM
- Panel "Sunucuyu guncelle" (POST /api/deploy): DEPLOY_SCRIPT_PATH = git pull, build, pm2. Veritabani tasimaz.
- export-superadmin-full-sql.cjs + psql: sadece PostgreSQL. Kod deploy etmez.

ONERILEN SIRA
1) Commit/push
2) Sema: migration (sunucuda MIGRATE_ON_DEPLOY=1 ile server-deploy.sh icinde migration:run - opsiyonel)
3) Kod: panel deploy veya server-deploy.sh
4) Veri (istege bagli): export -> yedek -> psql import
5) Import oncesi canlida pg_dump

EXPORT
  cd backend
  npm run export-superadmin-full-sql -- ../data-mirror.sql --skip-app-config

WINDOWS
  cd backend
  $env:DEPLOY_SSH_HOST="..."
  ..\backend\tools\deploy-local-to-prod.ps1
  Varsayilan: app_config SQL dislanir. Tam app_config: -IncludeAppConfig

SUNUCU server-deploy.sh
  MIGRATE_ON_DEPLOY=1  -> npm run migration:run (schema) sonra pm2

CANLI DEPLOY_SECRET (panel ile ayni)
- Yerel backend/.env icinde DEPLOY_* ayarla.
- Canliya aktar: backend/tools/sync-deploy-env-to-remote.ps1
  Ortam: DEPLOY_SSH_HOST, DEPLOY_SSH_USER, DEPLOY_SSH_KEY, DEPLOY_REMOTE_ROOT (opsiyonel)
  Sunucuda python3 gerekir; sonra pm2 restart uzaedu-api calisir.
