# TV slider görselleri

Bu klasöre slide görselleri ekleyebilirsiniz:

## Başarılarımız

- **Dosya adı:** `basari.jpg`
- **Kullanım:** Duyuruda `attachment_url` yoksa bu görsel kullanılır.
- **Örnek:** Kendi `basari.jpg` görselinizi indirip buraya kopyalayabilirsiniz.

## Ders Programı

- **Dosya adı:** `dersprogrami.jpg`
- **Kullanım:** Timetable slide'da admin tarafından eklenen görsel yoksa bu görsel kullanılır (orta alanda büyük).
- **Örnek:** Kendi `dersprogrami.jpg` görselinizi indirip buraya kopyalayabilirsiniz. Dosya yoksa görsel gizlenir.

Alternatif olarak admin panelinden **Toplu Ders Programı Ekle** veya duyuruya `attachment_url` (görsel linki) vererek ekleyebilirsiniz.
