## 5. Önerilen Çözüm Yolları

### Yol A: TYMM Taslak Planlarından İçe Aktarma (Tercih Edilebilir)

1. **Manuel indirme:** Kullanıcı tymm.meb.gov.tr’den RAR’ı indirir, açıp Excel/Word dosyasını seçer.
2. **Import:** Web-admin’de "MEB Taslak Planından İçe Aktar" butonu → dosya yükle → parse → `yillik_plan_icerik`’e yaz.
3. **Avantaj:** MEB’in resmi taslak planı; kazanım ve hafta dağılımı doğrudan MEB kaynağından.
4. **Teknik:** RAR içindeki Excel’i parse etmek için `xlsx` kütüphanesi; sütun eşleme (Hafta, Ünite, Konu, Kazanımlar, Ders Saati, Belirli Gün vb.).

### Yol B: mufredat.meb.gov.tr PDF’lerinden Kazanım Çıkarma

1. **PDF indirme:** ProgramDetay sayfasındaki PDF linkleri (örn. Coğrafya PDF).
2. **Parse:** PDF’ten metin çıkarma (pdf-parse, pdf2json vb.) → ünite ve kazanım bloklarını regex ile ayırma.
3. **Config güncelleme:** `curriculum-kazanimlar.ts`’e ekleme veya script ile güncelleme.
4. **Zorluk:** PDF yapısı derslere göre farklı; manuel doğrulama gerekebilir.

### Yol C: Hibrit (Mevcut + Geliştirme)

1. **Kazanımlar:** `curriculum-kazanimlar.ts` – mufredat PDF’lerinden manuel/yarı-otomatik alınan metinler.
2. **Plan yapısı:** TYMM taslak RAR → import script veya "Dosyadan içe aktar" UI.
3. **GPT:** Sadece kazanımı olmayan dersler için; veya TYMM’de olmayan eski programlar için fallback.
